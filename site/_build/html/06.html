6. 행 추가, 삭제 기능 만들기¶

지금까지 데이터를 출력하고 수정하는 작업을 처리해보았습니다. 행을 추가하고 삭제하는 기능을 추가해봅니다.

6.1. 버튼 추가하기¶

클릭 이벤트가 발생하면 그리드 API를 호출할 수 있도록 index.html 파일에 버튼 엘리먼트를 추가해봅니다.

<button id="add">추가</button>
<button id="remove">삭제</button>

6.2. 그리드 API 호출하기¶

클릭 이벤트 시 추가, 삭제에 대한 함수를 추가합니다. 각 함수에서는 동작에 따라 그리드 API를 호출하여 행이 추가되거나 삭제될 수 있도록 처리해봅니다.

// 행 추가
function addRow() {
    grid.appendRow([
        {
          date: '',
          category1: '',
          category2: '',
          payment: '',
          amount: ''
        }
    ]);
}

// 체크된 행 삭제
function removeRow() {
    var rowKeys = grid.getCheckedRowKeys();

    if (rowKeys.length) {
        grid.removeRow(rowKeys[0]);
    }
}

위에서 추가한 버튼에 클릭 이벤트를 바인딩하기 위해 다음 코드를 index.js 파일에 추가합니다.

document.getElementById('add').addEventListener('click', addRow);
document.getElementById('remove').addEventListener('click', removeRow);
